# sqlite3 local db demo

In the app, go to the "tags" page to add and delete tags, which are saved locally to sqlite.

**Check `lib/localdb.ts` for notes on design!!!**

We've tentatively decided not to use local databases for user data, but maybe this demo will be useful for the future for other features.

expo-sqlite docs: https://docs.expo.dev/versions/latest/sdk/sqlite
